# KOLForge

AI-driven one-stop virtual KOL marketing platform.

KOLForge keeps the original processing pipeline and upgrades the product positioning into a business-ready marketing workspace:
- Content Factory: upload or import long videos, generate high-performing short clips
- Distribution Center: publish to TikTok, Instagram Reels, and YouTube Shorts
- Virtual KOL Studio: generate AI actor videos and campaign assets

## Quick Start

1. Install Docker and Docker Compose
2. Start services:

```bash
docker compose up --build
```

3. Open dashboard: `http://192.168.1.54:5175`

## Core Value

- AI campaign production workflow in one place
- Multi-channel short-video distribution
- Virtual KOL content generation for product marketing
- Self-hostable architecture with extensible provider integrations

## Tech Stack

- Backend: FastAPI + Python
- Frontend: React + Vite + Tailwind
- Infra: Docker Compose
